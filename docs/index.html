Up and running in 5 minutes

No API keys. No third-party accounts. Uses your existing Google login.

1

Install the add-on

Install from the Google Workspace Marketplace directly into your Google Sheets.

2

Configure

Enter your Search Console site URL and GA4 Property ID. Choose a date range.

3

Authorize

Grant read-only access to your Search Console and Analytics data via Google OAuth.

4

Run the report

Click Run — all 7 tabs populate in seconds. Schedule it daily to keep data fresh automatically.

🔒 Your data never leaves Google. The add-on reads your Search Console and Analytics data using read-only OAuth scopes and writes it directly to your spreadsheet. No data is stored on external servers. Privacy Policy →
